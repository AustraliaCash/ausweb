Blockchain

The AustraliaCash blockchain is capable of high transaction speeds and short block generation times which help the network to support many transactions, large or small.

Wallet Security

Wallet encryption secures your wallet via password. The wallet.dat file can also be stored offline on a usb device.

Proof of Work

A Proof-of-Work (PoW) system (or protocol, or function) is an economic measure to deter denial of service attacks and other service abuses such as spam on a network by requiring some work from the service requester, meaning processing time by a computer.

Algorithm

AustraliaCash uses the Scrypt Algorithm to secure the blockchain. The algorithm was specifically designed to make it costly to perform large-scale custom hardware attacks by requiring large amounts of memory.
